# 空港出発案内板アプリ
Spring Bootを用いて空港の出発案内板を再現しました。<br>
出発案内板に出発便情報が時刻順に表示されます。<br>
出発便ごとに備考の切り替えができます。<br>
出発時刻の30秒前になると出発アナウンスが流れます。<br>
案内板右下の設定リンクを押すと、設定画面から出発便、航空会社、目的地の追加、編集、削除ができます。<br>

# 機能一覧
・出発便表示<br>
・出発便管理<br>
・航空会社管理<br>
・目的地管理<br>
・搭乗アナウンス（自動）<br>
